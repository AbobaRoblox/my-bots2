  ЧТО БЫЛО:
  ─────────
    ❌ Просто time.sleep(60)
    ❌ Нет логов
    ❌ Нет обработки ошибок
    ❌ Нельзя настроить

  ЧТО СТАЛО:
  ──────────
    ✅ Логирование в файлы (logs/shopbot_YYYYMMDD.log)
    ✅ Обработка всех ошибок
    ✅ Конфигурация через JSON файл
    ✅ Повторные попытки (retry 3 раза)
    ✅ Счетчик итераций
    ✅ Режим отладки (debug mode)
    ✅ Корректное завершение (graceful shutdown)
    ✅ Автоматические директории

  КАК ИСПОЛЬЗОВАТЬ:
  ─────────────────
    1. Откройте shopbot_config.json
    2. Измените настройки:
       - shop_url: "https://ваш-магазин.com"
       - check_interval: 60 (секунды)
       - max_retries: 3 (попытки)
       - debug: false (или true для отладки)
    3. Запустите: python shopbot.py
    4. Логи в папке: logs/

  ЧТО СОЗДАНО:
  ────────────
    📦 requirements.txt        - список Python зависимостей
    ⚙️ config.json             - настройки bot.py (создается автоматически)
    ⚙️ shopbot_config.json     - настройки shopbot.py (создается автоматически)

  1. REQUIREMENTS.TXT
  ───────────────────
     Что это:     Список Python библиотек для ботов
     Где:         В корне проекта
     Зачем:       - Легко установить все зависимости
                  - Docker автоматически установит все нужное
                  - Версионирование библиотек
     
     Как использовать:
     ┌─────────────────────────────────────────────────────────────┐
     │ pip install -r requirements.txt                              │
     └─────────────────────────────────────────────────────────────┘
     
     Как добавить новую библиотеку:
     1. Откройте requirements.txt
     2. Добавьте строку: requests>=2.31.0
     3. Установите: pip install -r requirements.txt

  2. CONFIG.JSON (для bot.py)
  ───────────────────────────
     Создается:   Автоматически при первом запуске bot.py
     Где:         В корне проекта
     
     Содержимое:
     ┌─────────────────────────────────────────────────────────────┐
     │ {                                                            │
     │     "check_interval": 60,     ← Интервал в секундах        │
     │     "bot_name": "Bot",        ← Имя для логов              │
     │     "debug": false            ← Режим отладки              │
     │ }                                                            │
     └─────────────────────────────────────────────────────────────┘
     
     Как изменить:
     1. Откройте config.json в Блокноте
     2. Измените значения
     3. Сохраните файл
     4. Перезапустите бота

  3. SHOPBOT_CONFIG.JSON (для shopbot.py)
  ───────────────────────────────────────
     Создается:   Автоматически при первом запуске shopbot.py
     Где:         В корне проекта
     
     Содержимое:
     ┌─────────────────────────────────────────────────────────────┐
     │ {                                                            │
     │     "check_interval": 60,        ← Интервал проверки        │
     │     "bot_name": "ShopBot",       ← Имя бота                │
     │     "debug": false,              ← Режим отладки            │
     │     "shop_url": "https://...",   ← URL вашего магазина      │
     │     "max_retries": 3             ← Попытки при ошибке       │
     │ }                                                            │
     └─────────────────────────────────────────────────────────────┘
     
     Важно:  Обязательно измените shop_url на адрес своего магазина!

  ШАГ 1: УСТАНОВИТЬ WSL 2
  ────────────────────────
    1. Откройте PowerShell от имени администратора:
       - Нажмите Win + X
       - Выберите "Windows PowerShell (администратор)"
    
    2. Выполните команду:
       ┌─────────────────────────────────────────────────────────────┐
       │ wsl --install                                                │
       └─────────────────────────────────────────────────────────────┘
    
    3. Перезагрузите компьютер

  ШАГ 2: СКАЧАТЬ DOCKER DESKTOP
  ──────────────────────────────
    1. Откройте браузер
    2. Перейдите на сайт:
       ┌─────────────────────────────────────────────────────────────┐
       │ https://www.docker.com/products/docker-desktop               │
       └─────────────────────────────────────────────────────────────┘
    
    3. Нажмите "Download for Windows"
    4. Скачается файл: Docker Desktop Installer.exe (~500 MB)

  ШАГ 3: УСТАНОВИТЬ DOCKER DESKTOP
  ─────────────────────────────────
    1. Запустите Docker Desktop Installer.exe
    2. В окне установки:
       ✅ Включите опцию "Use WSL 2 instead of Hyper-V"
    3. Нажмите "Ok"
    4. Дождитесь завершения установки
    5. Нажмите "Close and restart"
    6. Компьютер перезагрузится

  ШАГ 4: ПЕРВЫЙ ЗАПУСК
  ────────────────────
    1. После перезагрузки запустите "Docker Desktop"
    2. Примите лицензионное соглашение
    3. Войдите в аккаунт Docker (или нажмите Skip)
    4. Дождитесь запуска Docker (иконка в трее станет зеленой)

  ШАГ 5: ПРОВЕРКА
  ───────────────
    Откройте PowerShell и выполните:
    ┌─────────────────────────────────────────────────────────────┐
    │ docker --version                                             │
    │ docker-compose --version                                     │
    └─────────────────────────────────────────────────────────────┘
    
    Должно вывести версии Docker и Docker Compose

  ШАГ 6: ЗАПУСК БОТОВ
  ───────────────────
    Теперь можно запустить ботов в Docker!
    
    Дважды кликните на файл:
    ┌─────────────────────────────────────────────────────────────┐
    │ start_hosting.bat                                            │
    └─────────────────────────────────────────────────────────────┘
    
    Или в PowerShell:
    ┌─────────────────────────────────────────────────────────────┐
    │ cd "C:\гайд"                                                │
    │ docker-compose up -d                                         │
    └─────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│                        🎯 ПОЛЕЗНЫЕ КОМАНДЫ                                 │
└────────────────────────────────────────────────────────────────────────────┘

  DOCKER:
  ───────
    docker --version              # Проверить Docker
    docker-compose --version      # Проверить Compose
    docker-compose up -d          # Запустить ботов
    docker-compose down           # Остановить ботов
    docker-compose logs -f        # Посмотреть логи
    docker-compose ps             # Статус контейнеров
    docker-compose restart        # Перезапуск

  PYTHON:
  ───────
    python bot.py                 # Запустить bot.py
    python shopbot.py             # Запустить shopbot.py
    pip install -r requirements.txt   # Установить зависимости

  СКРИПТЫ:
  ────────
    start_hosting.bat             # Запустить Docker (Windows)
    stop_hosting.bat              # Остановить Docker (Windows)
    add_to_startup.bat            # Автозапуск на Windows
