  ШАГ 1: Запустите мастер настройки
  ──────────────────────────────────
    
    👉 Дважды кликните: START_CLOUD.bat
    
    Мастер поможет:
      - Проверить Git
      - Загрузить на GitHub
      - Открыть Railway
      - Задеплоить ботов


  ШАГ 2: Создайте GitHub репозиторий
  ───────────────────────────────────
    
    1. Откройте: https://github.com/new
    2. Имя: my-bots
    3. Нажмите "Create repository"
    4. Загрузите файлы (setup_github.bat или вручную)


  ШАГ 3: Задеплойте на Railway
  ─────────────────────────────
    
    1. Откройте: https://railway.app
    2. "Login with GitHub"
    3. "New Project" → "Deploy from GitHub repo"
    4. Выберите: my-bots
    5. "Deploy Now"
    6. ✅ Готово! Боты работают 24/7!

  🔄 Перезапуск:
     Railway → Settings → Restart

  ⏸️ Остановка:
     Railway → Settings → Delete Service

  ⚙️ Изменение кода:
     1. GitHub → Откройте bot.py
     2. Нажмите "Edit" (карандаш)
     3. Измените код
     4. "Commit changes"
     5. Railway автоматически обновится!

  📝 Изменение настроек:
     1. GitHub → Откройте config.json
     2. Нажмите "Edit"
     3. Измените параметры
     4. "Commit changes"
     5. Railway применит изменения!
